# 俄罗斯方块 (Tetris)

一个采用苹果风格设计的现代化俄罗斯方块游戏，具有流畅的动画效果和直观的用户界面。

## 游戏特色

- ? **苹果风格设计**：采用明亮、饱和的苹果风格配色方案
- ? **流畅的游戏体验**：平滑的动画和响应式控制
- ? **实时下一个方块预览**：提前了解下一个要出现的方块
- ? **可调节难度**：支持简单、中等、困难和专家四个难度级别
- ? **计分系统**：实时显示分数，消除行获得积分
- ? **经典玩法**：保留俄罗斯方块的经典游戏机制

## 如何运行

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！

## 游戏控制

- **← →**：左右移动方块
- **↓**：加速下落
- **↑**：旋转方块
- **空格**：直接下落到底部

## 难度级别

- **简单**：方块下落速度较慢，适合初学者
- **中等**：适中的下落速度，适合有一定经验的玩家
- **困难**：快速下落，需要快速反应
- **专家**：极快下落，挑战极限

## 计分规则

- 消除 1 行：100 分
- 消除 2 行：200 分
- 消除 3 行：300 分
- 消除 4 行：400 分

## 技术实现

- **HTML5 Canvas**：用于绘制游戏画面和下一个方块预览
- **JavaScript**：实现游戏逻辑和交互
- **CSS3**：采用苹果风格设计，包含渐变、阴影和圆角等现代化样式
